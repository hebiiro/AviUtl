  • SelectFavoriteFont - お気に入りフォント選択 - AviUtl-Plugin-SelectFavoriteFont
  • SelectEasing - イージング簡単選択 - AviUtl-Plugin-SelectEasing
  • SelectScene - シーン簡単選択 - AviUtl-Plugin-SelectScene
  • ShowWaveform - アイテム内音声波形 - AviUtl-Plugin-ShowWaveform
  • LevelMeter - レベルメーター - AviUtl-Plugin-LevelMeter

DarkenWindow - 黒窓 (くろまど)

AviUtl のスキン (UI のデザイン) を変更します。
AviUtl-Plugin-DarkenWindow

SplitWindow - SplitWindow

AviUtl の各ウィンドウを再帰分割されたペインに貼り付けます。UniteWindow の後継プラグインです。
AviUtl-Plugin-SplitWindow

UniteWindow - UniteWindow

AviUtl の各ウィンドウを一つのウィンドウにまとめます。
AviUtl-Plugin-UniteWindow

UniteWindowと出会ったおかげで僕のaviutlがいろんなプラグインでパワーアップしました(ついでにaviutlを一から入れなおした)

僕形から入ってそれで満足する人間だから編集してないけど(プラグイン入れる前の編集データ読み込んでるだけ) pic.twitter.com/xdFejSBZzK

SplitText - テキスト分解

テキストオブジェクトを一文字ずつに分解します。
AviUtl-Plugin-SplitText

DragFilter - フィルタドラッグ移動

設定ダイアログのフィルタをドラッグして移動できるようにします。
AviUtl-Plugin-DragFilter

CopyFilter - フィルタのコピペ

設定ダイアログのフィルタをコピペできるようにします。
AviUtl-Plugin-CopyFilter

SelectFavoriteFont - お気に入りフォント選択

よく使うフォントやお気に入りフォントを選択できるようにします。
AviUtl-Plugin-SelectFavoriteFont

SelectEasing - イージング簡単選択

イージング番号を画像から選択できるようにします。
AviUtl-Plugin-SelectEasing

SelectScene - シーン簡単選択

シーンを簡単に切り替えられるようにします。
AviUtl-Plugin-SelectScene

ShowWaveform - アイテム内音声波形

音声波形をアイテム内に表示します。
AviUtl-Plugin-ShowWaveform

LevelMeter - LevelMeter

レベルメーター (RMS とピーク) を表示します。
AviUtl-Plugin-LevelMeter

AutoVolume - オートボリューム

指定したレイヤーに音声アイテムがあるとき、自動で音量を調整します。
AviUtl-Plugin-AutoVolume

ConfigChecker - コンフィグチェッカー

動画を出力するときに設定が適正かどうかチェックします。
AviUtl-Plugin-ConfigChecker

AdjustLastFrame - 最終フレーム自動調整

最終フレームが最後のオブジェクト位置になるように自動的に調整します。
AviUtl-Plugin-AdjustLastFrame

AdjustDialogPosition - ダイアログ位置調整

ダイアログの表示位置がモニタからはみ出さないように調整します。
AviUtl-Plugin-AdjustDialogPosition

ObjectExplorer - オブジェクトエクスプローラ

簡易エクスプローラを表示してエイリアスファイルを使いやすくします。
AviUtl-Plugin-ObjectExplorer

OptimizeEditBox - エディットボックス最適化

主にエディットボックスの処理を最適化します。
AviUtl-Plugin-OptimizeEditBox

BigDialog - ダイアログ巨大化

ダイアログのサイズを調整してコントロールが見切れないようにします。
AviUtl-Plugin-BigDialog

ManyView - 複数ビューア

複数のフレーム画像ビューアを表示します。
AviUtl-Plugin-ManyView

SetZoom - ズーム変更

タイムラインのズーム値を指定された値に変更します。
AviUtl-Plugin-SetZoom

Toumei - 拡張編集透明化

拡張編集ウィンドウと設定ダイアログを透明化します。
AviUtl-Plugin-Toumei

BuildStairs - アイテムずらし

タイムラインのアイテムを整列します。
AviUtl-Plugin-BuildStairs

OutputSeq - 連番出力

連番画像を出力します。
AviUtl-Plugin-OutputSeq

SaveImage - 画像保存

現在のフレーム、または選択アイテムを画像として保存します。
AviUtl-Plugin-SaveImage
